## Hexatune
HexaTune is a lightweight system refinement engine designed to improve runtime behavior and device responsiveness through smart, adaptive tweaks. All adjustments are applied safely, work without root, and can target all apps or specific apps depending on user preference.


## Features
- Adaptive runtime tuning  
- Lightweight system refinement  
- Non-root safe operations  
- Per-app or global behavior control  
- Smooth and stable performance handling  
- Designed for everyday use and gaming environments  


## How It Works
HexaTune applies subtle runtime adjustments using safe Android mechanisms.  
All optimizations run in the background without modifying system partitions or requiring root access.


## Compatibility
- Android 10 or newer  
- Works on both rooted and non-rooted devices  
- Optimized for modern chipsets  


## Privacy
HexaTune does not collect or transmit personal data.  
All processing happens locally on the device.
